# Customer Support

!!! note "🚧 Coming soon"
    This harness is on the roadmap and not yet published. Want to help build it — or add your own? See [Add your agent](../contributing.md).

A tool-using support agent that **triages, retrieves, acts, and escalates** — drafting grounded, on-brand answers and handing off cleanly to a human when confidence drops.

## What it will do

- Triage an incoming ticket and retrieve grounded context from your knowledge base.
- Act through tools to resolve the request, drafting an on-brand reply.
- Escalate low-confidence or sensitive cases to a human with full context.

## Conductor features

| Feature | Role |
|---|---|
| ReAct loop | Reason → act → observe over the ticket and knowledge base |
| Tool tasks | Retrieval, ticketing, and action tools as first-class steps |
| Human-in-the-loop | Escalation gate — a durable signal task hands off to an agent |
